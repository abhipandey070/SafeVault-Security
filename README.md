# SafeVault Security Application

## Project Overview

SafeVault is a sample ASP.NET Core security project demonstrating secure coding practices, authentication, authorization, input validation, and security testing.

## Features

- User Authentication
- Role-Based Authorization
- Input Validation
- Password Hashing
- JWT Token Generation
- Security Testing
- Microsoft Copilot Assisted Development

## Technologies

- ASP.NET Core
- C#
- JWT Authentication
- Security Testing
- Microsoft Copilot

## Project Structure

```
Models/
    User.cs

Services/
    AuthService.cs

Security/
    InputValidator.cs
    PasswordHasher.cs
    JwtHelper.cs

Tests/
    SecurityTests.cs

Program.cs
README.md
```

## Copilot Assistance

Microsoft Copilot helped generate secure validation logic, authentication methods, authorization structure, password hashing examples, JWT implementation, security testing, and debugging suggestions. The generated code was reviewed and refined before submission.
